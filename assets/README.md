# Assets Folder

This folder contains the app's visual assets.

## Required Assets

To complete the app setup, you'll need to add these files:

1. **icon.png** (1024x1024 px) - App icon
2. **splash.png** (1242x2436 px) - Splash screen
3. **adaptive-icon.png** (1024x1024 px) - Android adaptive icon
4. **favicon.png** (48x48 px) - Web favicon

## Quick Setup

You can use any image editor or online tool to create these assets. Here are some recommendations:

- Use **Figma**, **Canva**, or **Adobe XD** for design
- For quick icons, use https://icon.kitchen/ or https://appicon.co/
- Make sure to use transparent backgrounds for icons
- Use the app's color scheme: Purple (#8B5CF6), Blue (#3B82F6), Green (#00FF87)

## Color Palette

- Primary: `#8B5CF6` (Purple)
- Secondary: `#3B82F6` (Blue)  
- Accent: `#00FF87` (Green)
- Error: `#FF4757` (Red)
- Warning: `#FFA500` (Orange)
- Background: `#0a0a0a` (Dark)

The app will work without these assets, but for production builds, you should add proper branding assets.
